HTML Style Attribute

HTML Style Attribute di definisikan menggunakan attribute <style> :

<tagname style="property:value;">
contoh :

<h4 style="background-color:red;">

This is heading 1

Text Color

Contoh dari teks color menggunakan <style> :

<h4 style="color:blue;">

This is blue

Text Font

Contoh dari teks font menggunakan property <font-family> :

<h4 style="font-family:verdana;">Ini font verdana<h4>
<h4 style="font-family:coursier;">Ini font coursier<h4>

Ini font Verdana


Ini font Courier

Text Size

Contoh dari teks size menggunakan property <font-family> :

<h4 style="font-size:250%;">Ini 250<h4>
<p style="font-size:160%;">Ini 160<p>

Ini 250


Ini 160

Text Align

Contoh dari teks align menggunakan property <text-align> :

<h4 style="font-align:left;">Ngiri.....<h4>
<h4 style="font-align:center;">..Nengah..<h4>

Ngiri....


..Nengah..
