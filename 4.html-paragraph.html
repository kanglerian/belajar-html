HTML Paragraph

HTML paragraph di definisikan menggunakan tag <p></p> :

<p>Paragraph.</p>
<p>Paragraph yang kedua.</p>

HTML Horizontal Rules

HTML Horizontal Rules dengan tag <hr> digunakan untuk memisahkan konten :

This is heading 1

This is some text.

This is heading 2

This is some other text.

<h4>This is heading 1</h4>
<p>This is some text.<p>
<hr>
<h5>This is heading 2</h5>
<p>This is some other text.</p>
<hr>

HTML Line Break

HTML line break dengan tag <br> digunakan untuk memisahkan memulai garis baru :

This is heading 1


This is some text.

<h4>This is heading 1</h4><br><p>This is some text.<p>
