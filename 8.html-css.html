Apa itu CSS?

Cascading Style Sheet (CSS) digunakan untuk mengatur layout webpage.
Dengan CSS kamu bisa mengatur warna, font, ukuran teks, jarak antar element dan posisi element, serta gambar latar belakang atau background warna, ukuran layar yang berbeda dan ukuran layar serta yang lainnya.

Menggunakan CSS

Ada 3 macam dalam penggunakan CSS :

  • Inline CSS

    • digunakan dengan menggunakan element style pada attribute element.

  • Internal CSS

    • digunakan dengan menggunakan element <style> di dalam <head>

  • Eksternal CSS

    • digunakan dengan menggunakan element <link>
