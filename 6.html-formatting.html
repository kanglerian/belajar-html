HTML Text Formating

Element-element yang biasa digunakan di dalam text adalah :

  • <b> - untuk Bold text
  • <strong> - untuk Important text
  • <i> - untuk Italic text
  • <em> - untuk Empashized text
  • <mark> - untuk Marked text
  • <small> - untuk Smaller text
  • <del> - untuk Deleted text
  • <ins> - untuk Inserted text
  • <sub> - untuk Subscript text
  • <sup> - untuk Superscript text
