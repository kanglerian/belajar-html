HTML Document

HTML Document di definisikan menggunakan tag dibawah ini :

<!DOCTYPE html>
<html>
<head>ini diisi oleh link</head>
<body>

<h1>Ini adalah Heading</h1>
<p>Ini diisi paragraph<p>
</body>
</html>

HTML Links

HTML links di definisikan menggunakan tag <a></a> :

<a href="www.instagram.com/kanglerian"> Profile Instagram </a>

HTML Paragraph

HTML paragraph di definisikan menggunakan tag <p></p> :

<p>Paragraph.</p>
<p>Paragraph yang kedua.</p>

HTML Images

HTML Images di definisikan menggunakan tag <img> :

<img src="photoku.jpg" alt="Ini keterangan photo width="104" height="142">
