Attribute href

Attribute href ini digunakan untuk mendefinisikan Hyperlink :

<a href="https://www.kanglerian.com">Website Kang Lerian</a>

Attribute src

Attribute src ini digunakan untuk mendefinisikan sumber atau embed image :

<img src="photo-kuring.jpg">

Attribute width dan height

Attribute width dan height ini digunakan untuk mendefinisikan Hyperlink :

<img src="photo-kuring.jpg" width="240px" height="240px">

Attribute alt

Attribute alt ini digunakan untuk mendefinisikan jika foto tidak muncul :

<img src="photo-kuring.jpg" alt="Ini foto kuring">

Attribute style

Attribute style ini digunakan untuk menambah style pada sebuah element, seperti warna, font, size dan yang lainnya :

<p style="color:red;">Ini adalah paragraph</p>

Attribute lang

Attribute lang ini digunakan mendeklarasikan bahasa pada website :

<!DOCTYPE html>
<html lang="en">
<body>

<h1>Ini adalah Heading</h1>
<p>Ini diisi paragraph<p>
</body>
</html>

Attribute tittle

Attribute tittle ini digunakan untuk memberikan informasi tambahan pada element :

<p tittle="Ini adalah paragraph">Ini adalah paragraph</p>
